# 步驟說明
## 編輯舊醫師資料
1. 首先！內容->文章->先篩選這2個項目，分類(你要編輯的醫療科)、標籤 (00-醫師)
2. 輸入醫師姓名，找到醫師資料後，點選進入。
3. 點擊 【Edit with Fronted Editor】，進入SP Page編輯器開始編輯。


## 編輯新醫師資料
1. 首先！內容->文章->先篩選這2個項目，分類(範本)、標籤 (00-醫師)
2. 搜尋 dr-cv-temple 的文章，並開啟它
3. 開啟上述範例檔案後，點選上方功能按鈕選【儲存並複製】
4. 更新文章名稱(醫師姓名)、別名(醫師編號drXXXXX)、狀態(已發布)，先按下儲存。
5. 切換【圖片與連結】分頁，更新該醫師的圖片編號，按下儲存。(這是設定科網醫師團隊的縮圖)
6. 切換【內容】分頁，點擊右下角的按鈕【Edit with Fronted Editor】，進入SP Page編輯器開始編輯。
7. 開啟新的醫師頁面後，點選Import Layout 匯入<a href="../dashboard.html">新醫師資料範本(先下載到電腦備用)</a>，開啟後再開始編輯醫師資料。
    - 記得要改「網路掛號」的連結(點選按鈕後，從路徑更新醫師編號 https://hlapp.tzuchi.com.tw/tchw/OpdReg/DtQuery.aspx?dtno=醫師編號&Loc=hl)
    - 記得要改醫師照片(點選圖片，從路徑更新醫師編號 images/00MD/dr/00000.png)
    - 醫師照片檔，公傳美編陳品璇後製圖片完成後，會上傳到目錄下(/images/00MD/dr/)。請醫務部助更新上述連結可至此找檔案。
