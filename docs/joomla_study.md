# Joomla Study Zone

慈濟醫院 Joomla 網站管理者學習資源。請依您的部門選擇對應的教學。

!!! warning "重要提醒"
    請勿從 Notes 複製內容貼到 Joomla 文章中。此舉會帶入複雜格式程式碼，拖慢網站速度，且院內檔案路徑在院外無法顯示（如圖片）。  
    **正確做法：** 先將圖片下載到電腦 → 上傳到 Joomla 系統 → 再插入文章內容。
